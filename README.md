# Program CRUD Sederhana Python

Program sederhana untuk melakukan operasi CRUD (Create, Read, Update, Delete) pada data buku.

## Fitur
- Tambah buku baru
- Lihat daftar buku
- Edit data buku
- Hapus buku
- Pencarian buku
- Penyimpanan data dalam file JSON

## Cara Menjalankan
```bash
python programsederhana.py
